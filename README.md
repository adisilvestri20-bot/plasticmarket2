# PlasticMarket — Landing Page

Landing page de alta conversión para PlasticMarket (matricería, inyección, extrusión y coextrusión de plástico), con catálogo de productos, compuesto antibacterial, proceso, certificaciones y formulario de cotización.

## Contenido
- `index.html` — página principal
- `support.js` — runtime necesario para que la página funcione
- `assets/` — logo, certificado y fotos de productos

## Cómo publicar (GitHub Pages)
1. Sube estos archivos a la raíz del repositorio.
2. En Settings → Pages, selecciona la rama principal como fuente.
3. La página quedará disponible en `https://<usuario>.github.io/<repo>/`.
